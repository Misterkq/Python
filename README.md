# 🐍 Meus Estudos de Python: Lógica e Algoritmos

Este repositório contém meus exercícios e mini-projetos desenvolvidos enquanto aprendo a linguagem Python. O foco aqui é dominar a lógica de programação e as estruturas fundamentais da linguagem.

## 🚀 O que tem por aqui?

### 📚 Exercícios de Lógica
Scripts focados em resolver problemas do dia a dia e cálculos matemáticos, como:
- **Cálculos e Conversões:** Aluguel de carros, conversores de moedas e medidas.
- **Matemática:** Cálculos de descontos, aumentos salariais e outros.
- **Estruturas Condicionais:** Verificação de médias e notas.

### 🎮 Mini-Projetos
- **Jogo da Adivinhação:** Um jogo interativo onde o usuário tenta acertar o número secreto. Utilizei a estrutura `match case` (Python 3.10+) e manipulação de entradas.

## 🛠️ Tecnologias Utilizadas
- Python 3.x
- Bibliotecas nativas (Math, Random)

## 💡 Objetivo
Fortalecer minha base em lógica para avançar em projetos de automação e back-end.
